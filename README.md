# CrimsonPro-Cyr-Typeface

A cyrillic version of the original [Crimson Pro](https://github.com/Fonthausen/CrimsonPro) typeface

Styles:
— Regular
— Italic
— Extrabold

Added language support:
- Russian
- Belorussian
- Ukrainian (missing a few glyphs) 

Future plans for languages:
- Serbian
- Bulgarian

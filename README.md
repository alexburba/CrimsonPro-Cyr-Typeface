# CrimsonPro-Cyr-Typeface

A cyrillic fork of the original [Crimson Pro](https://github.com/Fonthausen/CrimsonPro) typeface. The main focus is to add cyrillic -).

This typeface has next styles:
- Regular
- Italic
- Extrabold

Added language support:
- Russian
- Belorussian
- Ukrainian 

Future plans:
- Serbian (2025)
- Bulgarian (2025)
